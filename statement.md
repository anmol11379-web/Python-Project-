# PROBLEM STATEMENT 
Hospitals currently rely on manual, paper-based systems to manage critical information such as patient records, doctor and worker details, and appointments. This manual process is inefficient, and often leads to data loss, duplication, and delays in retrieving information. There is a need for an automated, reliable system that can store, update, search, and delete hospital data efficiently to ensure data integrity, reduce errors, and improve overall hospital management operations.
# SCOPE OF VARIABLE 
1. Automating hospital administrative tasks such as registering and managing patient, doctor, and worker information.
2. Maintaining patient details, medical problems, doctor departments, and worker responsibilities in a digital system.
3. Allowing secure login for authorized person to access and manage hospital records.
4. Providing functionalities to add, view, search, update, and delete records efficiently.
5. Simplifying hospital workflow by replacing manual paper-based record keeping with a computerized solution.
# TARGATED USER 
1. Hospital administrators and staff responsible for managing data.
2. Healthcare facilities such as hospitals, clinics, and outpatient centers seeking to automate their record-keeping.
3. Medical office managers.
# HIGH LEVEL FEATURES
1. User Authentication
2. Data Management
3. Storage
4. Search Functionality
5. Update Records
6. Record Deletion
7. Command-Line Interface
